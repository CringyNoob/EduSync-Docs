## Frequently Asked Questions

**Q: How often are notices updated?**  
A: EduSync checks the UIU website every hour for new notices.

**Q: Can I turn off notice notifications?**

**Q: What if a notice is not showing?**  
A: The system fetches from UIU's official notice page. If it's not there yet, it won't appear. Wait for next sync or refresh manually.

A: No, the page auto-updates. But you can manually refresh anytime.

## How to Rent Items

### Finding & Renting

1. **Browse RentHub** - Click "RentHub" in navigation
2. **Filter by Category** - Books, Electronics, Sports, etc.
3. **View Listing Details**:
   - Daily rental price
   - Item condition
   - Owner ratings
   - Availability calendar
4. **Request to Rent** - Select dates and submit request
5. **Owner Approves** - Wait for owner confirmation
6. **Pickup/Delivery** - Coordinate via chat
7. **Use & Return** - Enjoy your rental, return on time!

### Listing Your Items

1. **Go to "List Your Item"** section
2. **Fill Details**:
   - What you're renting (e.g., "Data Structure Textbook")
   - Daily rental price (be reasonable!)
   - Security deposit (optional but recommended)
   - Availability dates
   - Condition and any restrictions
3. **Upload Photos** - Show current condition
4. **Publish** - Your listing goes live
5. **Manage Requests** - Approve/decline rental requests
6. **Track Rentals** - Monitor active rentals in your dashboard

## Common Questions

**Q: What if the item gets damaged?**  
A: Security deposits cover minor damages. Major damages should be reported and resolved between renter and owner. Admins can mediate if needed.

**Q: Can I cancel a rental?**  
A: Yes, but cancellation policies vary by owner. Check listing details. Late cancellations may forfeit deposits.

**Q: What if someone doesn't return my item?**  
A: Contact the renter via chat first. If unresponsive, report to admins with rental details. Security deposits help protect owners.

**Q: How do payments work?**  
A: Payment is processed when rental is confirmed. Funds are held and released to owner after successful return. Security deposits are refunded if no damages.

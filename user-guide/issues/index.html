    How to Report an Issue¶

    Step-by-Step Guide¶

    1. Click "Report Issue" button
    2. Choose Category:
    3. Infrastructure
    4. WiFi/Internet
    5. Cafeteria
    6. Washrooms
    7. Security
    8. Library
    9. Laboratory
    10. Other
    11. Fill Details:
    12. Title - Short description (e.g., "Broken AC in Room 401")
    13. Description - Full details of the problem
    14. Location - Specific building/room
    15. Severity - Low, Medium, High, Critical
    16. Photos - Upload images (optional but helpful!)
    17. Submit - Your report is sent to admin immediately

    What Happens Next?¶

    graph LR
    A[You Submit Issue] --> B[Admin Reviews]
    B --> C[Status: Acknowledged]
    C --> D[Work In Progress]
    D --> E[Testing/Verification]
    E --> F[Resolved]
    F --> G[You Can Close]

    You'll receive notifications at each stage!

    Status What It Means What You Should Do
    üÜï New Just submitted Wait for admin review

    üîß In Progress Being worked on Check for updates
    üß™ Testing Fix is being verified Wait for completion
    ‚úÖ Resolved Issue is fixed Verify and close
    ‚ùå Rejected Not valid/duplicate See admin comment

    Tips for Effective Reporting¶

    üì∏ Take Photos¶

    Visual evidence helps admin understand the problem quickly.

    üìç Be Specific¶

    "Room 401, 4th Floor, CSE Building" is better than "CSE".

    ‚è∞ Mention Timing¶

    If the issue occurs at specific times, mention it.

    ü§ù Check for Duplicates¶

    Search if someone already reported it - upvote instead of duplicating.

    üìà Choose Correct Severity¶

    Be honest - don't mark everything as Critical.

    üí¨ Respond to Admin¶

    If admin asks for more details, respond promptly.

    Updates from Administration¶

    How to Respond¶

    1. Check notifications regularly
    2. Provide requested information promptly
    3. Verify when issue is marked resolved
    4. Close the issue if satisfied
